## 2.1 Верификация информационной базы и подтверждение достоверности источников.

Для обеспечения научной достоверности, репрезентативности и воспроизводимости результатов в рамках данной работы сформированная база данных, опирающаяся на официальные российские и международные базы данных.

Гидрометеорологические и климатические данные:

Данные по температуре воздуха, количеству осадков, температуре грунта, уровню снежного покрова взяты из открытых архивов ФГБУ “Всероссийский научно-исследовательский институт гидрометеорологической информации \- Мировой центр данных” (ФГБУ “ВНИИГИМЦД”, официальный портал [meteo.ru](http://meteo.ru)). Данный источник представляет из себя основную опорную базу климатических данных РФ, проходящий сквозную и экспертную верификацию и контроль качества в соответствии со стандартами Всемирной метеорологической организации (WMO)

Геокриологические данные (СТС/ALT):

Динамика мощности сезонно-талого слоя (Active Layer Thickness) получена из официальных реестров международной программы циркумполярного мониторинга вечной мерзлоты CALM (Circumpolar Active Layer Monitoring), координирумой Университетом Джорджа Вашингтона (база данных доступна по адресу [https://www2.gwu.edu/\~calm/data/data-links.htm](https://www2.gwu.edu/~calm/data/data-links.htm)). Рассмотрены данные, собранные методом зондирования на геокриологических стационарах Центральной Якутии.

Гидрологические данные:

Сведения о ежедневном уровне воды в сантиметрах над нулем графика поста взяты по ключевых створам Лено-Якутского бассейна. В рамках данной работы приняты к источнику сведения с портала [allrivers.info](https://allrivers.info/about). Далее для каждого гидропоста приведен первоисточник данных. Центр регистра и кадастра для гидропоста Киренск. Дальневосточное управление по гидрометеорологии для гидропостов Витим, Ленск, Олекминск, Покровск, Якутск, Табага, Сангар, Жиганск. Динамика прохождения весеннего половодья анализируется по суточным уровням воды. Ниже приведен график (рис. 2.1) амплитуд весенних пиков для визуального сопоставления гидрологических профилей.

Событийный архив Чрезвычайных ситуаций:

На данный момент события ЧС берутся из той же базы данных, что и гидрометеорологические и климатические данные: ФГБУ “ВНИИГИМЦД” (ссылка на источник данных [http://meteo.ru/data/adverse-weather-conditions/](http://meteo.ru/data/adverse-weather-conditions/)). Данные приводятся для всей территории России по 22 типам ЧС, среди них отбираются данные по Чрезвычайным событиям: “затор”, “паводок”, “половодье” на территории бассейна Лены.

## 2.2 Обоснование пространственно-временных границ расчетного контура

На этапе разведочного анализа данных были полностью исключены станции, не относящиеся к руслу реки Лена, в результате чего были оставлены 4 репрезентативные станции: г. Киренск, г. Олекминск, г. Якутск, г. Жиганск.

Аналогичное сужение приведено для станций на геокриологическом блоке станций CALM: из широкого перечня региональных площадок выбраны две, локализованные в районе г. Якутска: R42 и R43. Ограничение выборки данными объектами продиктовано необходимостью анализа и сопоставления деградации вечной мерзлоты в условиях различной антропогенной нагрузки в пределах одной макроклиматической зоны:

- Площадка R42 характеризуется урбанистическим ландшафтом. Развернута непосредственно в черте г. Якутска. Позволяет детектировать динамику СТС в условиях нарушенного напочвенного растительного покрова, теплового загрязнения атмосферы, техногенного пресса городских инженерных коммуникаций и фундаментов застройки. Данные площадки R42 критически важны для планирования превентивных мер МЧС при угрозе разрушения жилого и промышленного фондов города.  
- Площадка R43 характеризуется естественным ландшафтом. Данная площадка вынесена за пределы агломерации и функционирует в условиях природного таежно-атласного ландшафта, характерного для Центральной Якутии. Показатели мощности сезонно-талого слоя грунта на этой площадке служат базовой линией, свободной от урбанистических искажений.

Для большинства гидрометеорологических, климатических данных в исходных архивах сохранен ряд с конца XIX века по 2025 год, для гидрологических данных сохранен ряд с 2001 по 2026 год (исключением является гидропост г. Киренск, где данные сохранены до 2024 года). Для глубины сезонного протаивания грунта на целевых площадках CALM приведен ряд исторических значений ALT с 2008 по 2023 года, ввиду чего было принято решение усечь исторические массивы, приведя расчетный контур работы к строго фиксированному временному диапазону с 2008 по 2023 года. Поскольку методология исследования предполагает жесткой синхронизации всех предикторов в едином временном окне, доступность данных программы CALM выступила лимитирующим фактором.

На основе проведенного разведочного анализа данных было выбрано два уровня временной дискретизации: суточный уровень и годовой уровень. Суточный шаг данных рассматривается для оперативных данных: температура воздуха, количество осадков, температура грунта, уровень снежного покрова, уровень воды на гидропостах. Годовой шаг рассматривается для инерционных мерзлотных процессом (данные по сезонно-талому слою) и сценарного планирования. Месячное агрегирование исключено из данной схемы ввиду потери,напрямую связанных с ЧС, коротких критических эпизодов таких, как суточные пики уровня воды, экстремальные осадки, аномально теплые дни и оттепели, при переходе к месячным суммам и средним.

## 2.4 Методика LC-кривых и обоснование операционного окна упреждения.
