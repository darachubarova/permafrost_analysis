# Паспорт математической модели (в стиле академического раздела ВКР)

## 1. Общая характеристика

### 1.1. Объект исследования

Криолитозона и гидрологическая система бассейна реки Лены на территории Республики Саха (Якутия) в условиях современного климатического изменения.

### 1.2. Предмет исследования

Количественные зависимости между аномалиями метеорологических и мерзлотно-гидрологических индикаторов и частотой гидрологических чрезвычайных ситуаций (половодье, паводок, затор) в пределах анализируемого периода.

### 1.3. Цель

Разработать интерпретируемую математико-статистическую модель, обеспечивающую:

- выявление аномальных состояний природной среды;
- оценку краткосрочных и межгодовых рисков ЧС;
- перевод результатов оценки риска в сценарии реагирования для органов управления и подразделений МЧС.

### 1.4. Научная гипотеза

Годы с повышенной частотой гидрологических ЧС статистически связаны с комплексом предшествующих аномалий термического режима, снегонакопления, осадков и характеристик сезонного протаивания, причем часть индикаторов действует синхронно, а часть имеет лаговый эффект (N -> N+1).

## 2. Исходные данные и границы применимости

### 2.1. Пространственные границы

Основной расчетный профиль вдоль р. Лены:

- Киренск;
- Олёкминск;
- Якутск;
- Жиганск.

Фоновые станции Верхоянск и Оймякон используются для сравнительного климатического контекста, но исключаются из событийной увязки аномалий с ЧС.

### 2.2. Временные границы

Базовый период анализа: 2008-2023 гг.

Для корректного учета зимних процессов используется сквозная сезонность (ноябрь-декабрь предыдущего года относятся к зимнему сезону последующего паводкового года).

### 2.3. Информационные источники

Используются шесть групп данных:

1. Воздух и осадки (суточные ряды TTTR).
2. Температура почвы по глубинам.
3. Снежный покров (высота снега).
4. Уровень воды по гидропостам Лены.
5. Данные CALM (ALT/СТС, площадки R42/R43 для Якутска).
6. Архив ЧС (дата начала/окончания, тип, локация).

## 3. Формализация индикаторов

### 3.1. Годовые температурные индексы

Для года y рассчитываются:

`TDD(y) = sum(max(Tmean(d), 0)), d in Apr..Oct`

`FDD(y) = sum(max(-Tmean(d), 0)), d in winter_season(y)`

где `winter_season(y)` - физический зимний сезон, объединяющий ноябрь-декабрь года y-1 и зимне-весенние месяцы года y.

### 3.2. Производные климатические признаки

- `winter_thaw_days`: число зимних дней при `Tmean > 0`;
- `max_daily_precip`: годовой максимум суточных осадков;
- `spring_precip`: сумма осадков за апрель-май;
- `max_snow_depth`: максимум высоты снежного покрова в сквозном сезонном окне.

### 3.3. Критерий Rain-on-Snow (ROS)

Событие ROS фиксируется при выполнении системы условий:

- `month in [10, 11, 12, 1, 2, 3, 4]`;
- `Tmean > 0`;
- `precip_mm > 1`;
- `snow_depth_cm > 5`.

Сезонная привязка: октябрь-декабрь N относятся к сезону N+1; январь-апрель N - к сезону N.

### 3.4. Мерзлотные показатели

Вычисляются:

- `MAGST` (среднегодовая температура грунта);
- `TTOP` (температура на кровле мерзлоты);
- `MAAT` (среднегодовая температура воздуха);
- `SO = MAGST - MAAT`;
- `TO = TTOP - MAGST`;
- `N_t = TDD_soil / TDD_air`;
- `N_f = FDD_soil / FDD_air`.

Для Якутска в сценарной оценке дополнительно используется `ALT_z`.

## 4. Методика выявления аномалий

Для каждого индикатора реализован ансамбль из трех критериев.

### 4.1. Глобальная Z-нормировка

`z = (x - mean(x)) / std(x)`

### 4.2. Перцентильная диагностика хвостов

- умеренная аномалия: `x >= P95`;
- экстремальная аномалия: `x >= P99`.

### 4.3. Локальная временная норма

`rolling-window = 7 лет`, `min_periods = 4`, норма строится только по прошлым годам (`shift(1)`).

Для индикаторов типа "чем меньше, тем хуже" применяется инверсия перед расчетом (в частности, для `FDD`), что обеспечивает единую направленность шкалы риска.

Итоговые флаги `moderate` и `extreme` формируются логическим объединением результатов трех критериев.

## 5. Оперативный блок (день/месяц)

### 5.1. Суточная детекция

Суточная климатология формируется по дню года в окне +/-7 дней. Далее детектируются сигналы:

- аномально теплый день (`tmean_z >= 2`);
- зимняя/весенняя оттепель;
- экстремальные суточные осадки (`precip_mm >= P95`);
- резкое потепление (`delta_tmean_1d >= 8 C`).

На основе комбинации сигналов вычисляются `risk_points` и присваивается `severity`.

### 5.2. Месячная детекция

Месячные агрегаты сопоставляются с нормой соответствующего календарного месяца по другим годам:

- аномально теплый месяц;
- аномально влажный месяц;
- повышенное число зимних оттепелей.

## 6. Блок сопряжения с ЧС

### 6.1. Синхронная схема

Индикаторы года N сопоставляются с числом ЧС года N.

### 6.2. Лаговая схема

Индикаторы года N сопоставляются с числом ЧС года N+1:

`year_chs = year_indicators + 1`.

### 6.3. Контрастный анализ

Годы группируются как:

- `safe` (`chs_count = 0`),
- `norm` (`chs_count = 1`),
- `spike` (`chs_count > 1`).

Для каждого нормированного индикатора рассчитывается контраст:

`delta_z = mean(z | spike) - mean(z | safe)`.

Интерпретационно `delta_z` характеризует направление и силу сдвига индикатора при переходе от безопасного режима к всплеску ЧС.

### 6.4. Событийная увязка (суточный масштаб)

Для каждой даты аномалии выполняется поиск связанных ЧС в окне `-3...+30` суток с разбиением лагов на интервалы:

- 0-3,
- 4-7,
- 8-14,
- 15-30 суток.

## 7. Гидрологический подмодуль

По гидропостам рассчитываются:

- весенний максимум уровня (`spring_peak_cm`),
- амплитуда весеннего подъема,
- максимальная 7-суточная скорость подъема,
- число дней выше локального `P90/P95`,
- Z-оценки перечисленных характеристик.

Дополнительно оцениваются:

- лаги дат пика относительно Якутска,
- доля постов с гидроэкстремами,
- корреляция доли гидроэкстремов с годовым числом ЧС.

## 8. Сценарная классификация реагирования

Классификация выполняется по совокупности флагов `n_moderate`, `n_extreme` и, при наличии, `ALT_z`.

Правила:

- критический сценарий: `n_extreme >= 3` или (`n_extreme >= 2` и `|ALT_z| >= 3`);
- высокий сценарий: `n_moderate >= 2` и `n_extreme >= 1`;
- умеренный сценарий: `n_moderate >= 1`;
- базовый сценарий: в остальных случаях.

Для каждого класса заранее определен набор управленческих мер (мониторинг, оповещение, готовность сил, эвакуационные процедуры).

## 9. Научная новизна и практическая значимость

### 9.1. Научная составляющая

Выполнена единая формализация многомасштабной системы индикаторов (климат, мерзлота, гидрология) с разделением синхронных и лаговых эффектов в рамках одного вычислительного контура.

### 9.2. Практическая составляющая

Результаты моделирования трансформируются в операционные сценарии реагирования, что обеспечивает прямую применимость методики в задачах планирования мер по предупреждению и минимизации последствий ЧС.

## 10. Ограничения исследования

1. Пороговые правила имеют экспертную природу (rule-based) и не являются результатом параметрической оптимизации на обучающей выборке.
2. Интервал 2008-2023 гг. ограничивает статистическую устойчивость оценки редких экстремумов.
3. Качество выводов зависит от полноты базы ЧС и плотности сети наблюдений.
4. Модель выявляет статистические и лаговые зависимости, но не является строгой каузальной моделью в формально-эконометрическом смысле.

## 11. Итоговая формулировка для включения в ВКР

Разработанная математическая модель представляет собой методически согласованный и воспроизводимый инструмент анализа риска гидрологических ЧС в криолитозоне Якутии. Модель сочетает физически интерпретируемые индикаторы, статистически корректную процедуру выявления аномалий и управленчески ориентированную систему сценарной классификации, что обосновывает ее использование в аналитических и прикладных разделах выпускной квалификационной работы.
