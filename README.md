# Permafrost Analysis

This repository contains tools and scripts for analyzing permafrost data. The aim is to facilitate research and provide insights into permafrost conditions and dynamics.

Цель работы: найти взаимосвязь между показателями деградации вечной мерзлоты, текущими метеорологическими, гидропоказателями и чрезвычайными происшествиями на реке Лена, провести сценарный анализ с рекомендациями реакции государственных органов и спасательных служб
